## Command Fixture Matrix & Test Results

The new parser supports all target ecosystems. Unit tests verify the following parsing behaviors:

1. **npm**: `npm install axios lodash --save-dev -g` -> parsed `axios`, `lodash`
2. **yarn**: `yarn add lodash` -> parsed `lodash`
3. **pip**: `pip install requests==2.31.0` -> parsed `requests`
4. **poetry**: `poetry add requests` -> parsed `requests`
5. **go**: `go get github.com/foo/bar@v1.0.0` -> parsed `github.com/foo/bar`
6. **cargo**: `cargo add serde --features derive` -> parsed `serde`

All tests pass.
